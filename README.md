# Credit Card Approval Prediction

## Overview
This project is a Machine Learning web application that predicts whether a credit card application is likely to be approved based on user-provided information. The application uses a trained ML model and a Flask backend with an interactive web interface.

## Features
- Predicts credit card approval status
- User-friendly web interface
- Machine Learning-based prediction
- Responsive dashboard
- Input validation
- Fast prediction results

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Scikit-learn
- Pandas
- NumPy

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open the project folder

```bash
cd Credit-Card-Approval-Prediction
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
## How to Use

1. Open the application.
2. Enter the required applicant details.
3. Click the Predict button.
4. View the prediction result.

## Machine Learning Model

The model predicts whether a credit card application will be approved using applicant information after preprocessing and training.
